Woocommerce Product Dropdown field for Ninja form.

== Description ==

This plugins adds Woocommerce Product dropdown selection field for Ninja forms, useful for support form.

Note: This plugin is not compatible for Ninjaform 3.x, we are working to release a new version.

Our other plugins for Ninja Form plugin

* [Woo Order Field for Ninja Forms](https://wordpress.org/plugins/woo-order-field-for-ninja-forms/)

== Installation ==

This section describes how to install the plugin and get it working.

Upload woo-product-field-for-ninja-forms.zip file through WordPress admin, activate, go to ninja forms, use this addon,

or

Go to plugins menu from wordpress dashboard, search this plugin with keyword "woo-product-field-for-ninja-forms", install and activate

or

FTP:
1. Upload `woo-product-field-for-ninja-forms` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
